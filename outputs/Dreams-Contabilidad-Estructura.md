# Dreams Contabilidad: estructura inicial

## Objetivo

Crear un sistema contable web para una agencia de publicidad en Ecuador con productos y servicios generados por maquinaria láser CO2.

## Prototipo creado

- Aplicación web con interfaz moderna y responsive.
- Login con usuario y contraseña de demostración.
- Panel financiero con ventas, gastos, utilidad bruta e IVA.
- Registro de ventas, gastos, productos y clientes.
- Reportes básicos para IVA cobrado, IVA pagado y saldo referencial.
- Configuración editable del negocio, ciudad, RUC, actividad e IVA.

## Usuario inicial

- Correo: `admin@dreams.ec`
- Contraseña: `Dreams2026!`

## Modelo contable base

- Moneda: USD.
- IVA inicial: 15%.
- Ventas: subtotal, IVA y total.
- Gastos: subtotal, IVA y total.
- IVA referencial por declarar: IVA cobrado en ventas menos IVA pagado en gastos.
- Utilidad bruta: subtotal de ventas menos subtotal de gastos.

## Módulos pensados para crecimiento

- Usuarios reales y roles.
- Inventario avanzado.
- Facturación electrónica.
- Retenciones.
- Cuentas por cobrar y pagar detalladas.
- Reportes mensuales y anuales.
- Exportación a Excel/PDF.
- Integración con estadísticas del Banco Central del Ecuador.
- Configuración tributaria actualizable desde panel administrativo.

## Fuentes base recomendadas

- SRI: https://www.sri.gob.ec/
- BCE estadísticas económicas: https://www.bce.fin.ec/estadisticas-economicas/
